# Многоконтейнерное приложение

Данное приложение состоит из сервисов Nginx, Redis и MySQL, настроенных с использованием Docker Compose.

## Файлы в проекте:
- `docker-compose.yml`: Конфигурационный файл для запуска всех сервисов.

## Инструкция по запуску приложения:

1. **Установите Docker и Docker Compose**:
   Убедитесь, что Docker и Docker Compose установлены на вашем компьютере.

   - [Инструкция по установке Docker](https://docs.docker.com/get-docker/)
   - [Инструкция по установке Docker Compose](https://docs.docker.com/compose/install/)

2. **Распакуйте ZIP-файл**:
   Распакуйте файл `multi-container-app.zip`.

3. **Перейдите в папку проекта**:
   В терминале перейдите в папку с распакованными файлами:
   ```bash
   cd multi-container-app
   ```

4. **Запустите приложение**:
   Используйте команду Docker Compose для запуска:
   ```bash
   docker-compose up -d
   ```

   Это запустит веб-сервер Nginx, Redis и MySQL.

5. **Проверьте, что сервисы запущены**:
   Убедитесь, что контейнеры работают:
   ```bash
   docker-compose ps
   ```

6. **Доступ к веб-серверу**:
   Откройте браузер и перейдите по адресу:
   ```
   http://localhost:8080
   ```

7. **Остановка приложения**:
   Чтобы остановить контейнеры, выполните:
   ```bash
   docker-compose down
   ```

## Примечания:
- Данные MySQL сохраняются в volume `mysql_data`.
- Все сервисы соединены через сеть `mynetwork`.
